Travel guiding web app that contain sections like blogs,tour packages
and tour best offers. All data is hooked up with NoSQL database MongoDB.
APIS are designed with GraphQL and Apollo client is used as a GraphQL
client for ReactJS. No UI library is used throughout just the pure CSS.
For Carousel purposes, Libraries like react slick, swipeJS are given 
preferences,react reposive carousel. Foe blogs, GraphQL subscription
type has been used to provide real-time features for user added comments on individual blogs
